# pgo-git-start

To repozytorium zawiera ćwiczenie 1 z przedmiotu PGO.
Celem ćwiczenia jest nauka podstaw Git i GitHub:
- tworzenie repozytorium
- commity
- gałęzie
- merge

## Struktura repozytorium

- README.md
- src/Main.java

## Podstawowe polecenia Git

- git status
- git add
- git commit
- git push
- git checkout
- git merge
